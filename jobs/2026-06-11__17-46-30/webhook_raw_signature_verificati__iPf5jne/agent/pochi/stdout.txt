### Script (`f/zealt_zr-ipf5jne/sig_verify.py`)

Uses only stdlib `hmac` + `hashlib.sha256` and `wmill.get_variable` to read the secret:

```python
import hmac, hashlib, wmill

def main(raw_string: str = "", x_signature: str = "") -> dict:
    secret = wmill.get_variable("f/zealt_zr-ipf5jne/webhook_secret")
    expected_mac = hmac.new(
        secret.encode("utf-8"), raw_string.encode("utf-8"), hashlib.sha256
    ).hexdigest()
    if hmac.compare_digest(expected_mac, x_signature.lower()):
        return {"valid": True}
    return {"valid": False}
```
